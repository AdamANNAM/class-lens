# Changelog

## [0.1.0] - 2026-04-04

### Added
- Inline annotations showing `className` and `class` attribute values next to closing tags
- Support for JSX, TSX, HTML, Vue, Svelte, and Astro files
- Decoration and inlay hint rendering modes
- Configurable truncation (character/word, start/end)
- Customizable prefix, font style, and opacity
- Per-language exclusion list
